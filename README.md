# SRLINES Blog Content

This repository contains a Hugo-ready content structure for `https://blog.srlines.net`.

## What's included
- `config.toml` basic Hugo site configuration.
- `content/_index.md` homepage intro.
- 52 SEO-focused blog posts in `content/posts/` tailored to SRLINES and WhatsApp automation use cases.

## Run locally
```bash
hugo server -D
```

## Deploy
Publish this repository with GitHub Pages (via GitHub Actions) and map `blog.srlines.net` in DNS.
