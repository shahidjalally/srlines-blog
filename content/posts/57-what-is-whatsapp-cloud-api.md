WhatsApp Cloud API is Meta's hosted API that lets businesses send and receive WhatsApp messages programmatically without self-hosting WhatsApp infrastructure.

## Key capabilities

- Send approved template messages.
- Receive inbound customer messages in real time.
- Build chatbot, CRM, and notification workflows.

For most teams, Cloud API is the fastest path to production-grade WhatsApp automation.
